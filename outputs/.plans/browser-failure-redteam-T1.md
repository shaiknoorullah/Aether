# T1 Brief: Browser Failures & Post-Mortems

## Focus
Arc Browser's wind-down and other ambitious browser failures/stalls (2020-2026).

## Key Questions
1. What happened with Arc Browser? Timeline of rise and decline. What did The Browser Company officially say? What did they pivot to? User/community reaction.
2. What happened to Mighty Browser, SigmaOS, Sidekick Browser, Beam Browser, Station app? For each: when launched, what they promised, why they failed or stalled, any post-mortem statements.
3. Vivaldi — still alive but what's their growth trajectory? User count over time if available.
4. What common patterns emerge across these failures?

## Instructions
- Use web search extensively. Search for each browser by name + "shutdown", "pivot", "post-mortem", "lessons", "what happened".
- For Arc specifically, search for "The Browser Company pivot", "Arc Browser wind down", "Arc Browser 2025 2026", "Josh Miller Arc".
- Capture direct quotes from founders/teams where possible.
- Record every source URL.
- Write findings to `outputs/.drafts/browser-failure-redteam-research-failures.md`.
- Do NOT fetch PDF URLs. Prefer HTML articles, blog posts, press coverage.
- If a source is paywalled, note it and cite the URL anyway.

## Output Format
Markdown with sections per browser, direct quotes, source URLs inline.
