# Researcher Brief T2: Switching Intent + AI-Browser Adoption

## Objective
Find data on (a) user intent to switch browsers and reasons for switching, and (b) adoption rates/download counts/DAU for AI-native browsers.

## Key Questions

### Switching Intent
1. What share of users plan to switch browsers in 2025–2026? Any surveys?
2. What are the top reasons users switch (privacy, AI features, performance, etc.)?
3. Are there age/demographic differences in switching intent?

### AI-Browser Adoption
1. Arc browser: download counts, DAU, growth data (The Browser Company announcements)
2. Brave: current user count (MAU/DAU), Leo AI feature adoption
3. Edge Copilot: usage statistics from Microsoft
4. Opera Aria: user numbers
5. Perplexity browser (if launched): user data
6. Any analyst estimates of AI-browser market segment size

## Sources to Target
- Browser vendor press releases and blog posts
- Tech press coverage (The Verge, TechCrunch, Ars Technica)
- User surveys (Stack Overflow Developer Survey, browser vendor surveys)
- App store download data (if available)
- Analyst firm coverage (Gartner, IDC, Forrester snippets)
