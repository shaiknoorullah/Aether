## Evaluation Dimensions

1. **AI Features** — chat, agents, page summarization, auto-fill, tab management, search
2. **Architecture** — model hosting (local/cloud/hybrid), browser engine (Chromium/Firefox/WebKit/custom)
3. **Model Strategy** — proprietary model, third-party API (GPT/Claude/Gemini), open-source, multi-model
4. **Pricing** — free tier, subscription, usage caps
5. **Privacy Posture** — data collection, on-device processing, telemetry, privacy policy commitments
6. **Platform Support** — macOS, Windows, Linux, iOS, Android
