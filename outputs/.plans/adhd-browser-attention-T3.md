# Researcher Brief T3 — Tooling Landscape

**Topic:** Existing browser tools, extensions, and OS-level software designed for ADHD/neurodivergent users — features and effectiveness evidence

## Your Job
Research the landscape of existing tools:

1. Browser extensions: Freedom, Cold Turkey Blocker, StayFocusd, LeechBlock, OneTab, Workona, Toby, Session Buddy, Momentum, Forest — what features do they offer, what evidence (if any) supports effectiveness?
2. Built-in browser focus features: Chrome Focus Mode, Firefox Containers, Brave Shields, Arc spaces/profiles — document what each does
3. OS-level tools: macOS Focus Modes, Windows Focus Assist, Android/iOS digital wellbeing — any studies on effectiveness?
4. Reading/distraction-reduction: Firefox Reader View, Safari Reader, Readwise Reader, Instapaper — ADHD relevance
5. AI-integrated browser features: current state in 2025-2026 — which browsers have AI assistants, what do they do for focus/EF support?
6. Tab management: research on tab hoarding behavior in ADHD users; tools that address it
7. Any RCT or controlled study showing a digital focus tool reduced ADHD symptoms or improved task completion
8. User reviews / community discussion from ADHD communities (r/ADHD, ADDitude magazine, etc.) — what tools do people actually use?
9. Pricing models and accessibility barriers — free vs. paid tools

## Output
Write everything to: `outputs/.drafts/adhd-T3-research.md`

Include:
- Tool name, URL, key features
- Any effectiveness evidence with citation
- Source title, year, URL for every claim
- Confidence level per claim
- Gaps: features users want but no tool provides

Use web search. Prefer HTML pages. Do NOT fetch PDFs.
