# Comparison Plan: Privacy Browsers 2026

**Slug:** `privacy-browsers-2026`
**Date:** 2026-06-10

## Subjects
1. Brave Browser
2. LibreWolf
3. Mullvad Browser
4. Tor Browser
5. DuckDuckGo Browser (Desktop)

## Dimensions to Evaluate

| Dimension | Sub-points |
|-----------|-----------|
| **Blocking Engine** | Adblock/tracker list engine, default shields, custom lists support |
| **Anti-Fingerprinting** | Canvas/WebGL/audio/font randomization, user-agent spoofing, Letterboxing, RFP |
| **Telemetry Stance** | What data is collected, opt-in/opt-out, update pings, crash reports |
| **Default Search & Monetization** | Default search engine, revenue model, search deal disclosures |
| **User Base** | Estimated active user count, growth trend, primary demographic |

## Evaluation Criteria per Source
- **Source**: URL or document
- **Key Claim**: What it asserts
- **Evidence Type**: Official docs, audit, benchmark, community report, media
- **Caveats**: Bias, staleness, methodology gaps
- **Confidence**: High / Medium / Low

## Expected Output Structure
1. Executive summary paragraph
2. Comparison matrix (one row per browser, columns per dimension)
3. Agreement / Disagreement / Uncertainty analysis
4. Radar chart (qualitative scoring, 5 axes) via pi-charts
5. Mermaid architecture diagram for anti-fingerprint pipeline comparison
6. Sources section with direct URLs

## Source Strategy
- Official browser documentation / release notes / privacy policies
- Independent audits (Arkenfox wiki, privacytests.org, Privacy Guides)
- News/press releases (2025–2026)
- Web search for user-base estimates and monetization disclosures
- GitHub repos for telemetry configuration evidence

## Quality Gates
- Every claim linked to ≥1 explicit source URL
- Disagreements flagged explicitly
- Uncertain/unverified items marked `[UNVERIFIED]`
