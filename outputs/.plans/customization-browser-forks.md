# Comparison Plan: Customization-Focused Browser Forks 2026

**Slug**: `customization-browser-forks`
**Date**: 2026-06-10

## Scope

Browsers: Zen, Floorp, Waterfox, LibreWolf, Mullvad, Min, Thorium

## Dimensions

1. **Engine Base** — which upstream (Firefox/Gecko, Chromium/Blink, other), version lag
2. **Theming / Ricing Capability** — CSS injection, userChrome, sidebar mods, themes, UI customization depth
3. **Privacy Defaults** — telemetry, fingerprinting, first-party isolation, default search, tracking protection
4. **Update Cadence** — release frequency, upstream lag, maintenance status
5. **Core Differentiators** — what makes each browser uniquely positioned

## Sources to Pull

For each browser:
- Official website / docs
- GitHub releases page (cadence)
- Privacy policy / about:config defaults
- Community discussion (Reddit, GitHub issues for theming)

## Expected Output Structure

- Summary table (7 browsers × 5 dimensions)
- Per-browser section with claim, evidence, caveats, confidence
- Mermaid diagram: engine lineage tree
- Disagreement / uncertainty flags
- Sources section

## Evidence Types

- Direct: official docs, GitHub, release notes
- Indirect: user reports, community wikis
- Inferred: from upstream defaults if fork-specific docs are absent
